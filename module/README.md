# Coupon Hint

## Description
**Coupon Hint** is an OpenCart module that suggests available coupons to your customers for eligible products. It can also help you comply with supplier MAP policies while maintaining competitive prices.

Compatible with OpenCart 3.x (PHP ≥ 7.x).

Note: The source code is encrypted. For purchase and licensing information, see the OpenCart Marketplace link in the **Download** section.

## Features
* Automatically detects applicable coupons for a product
* Considers both product-specific and category-based coupons
* Selects the best or worst coupon based on configurable mode
* Multilingual support with translation-ready interface.
* No core file modifications.
* Seamless compatibility with the **Journal** theme.

## Restrictions
This module does not work and is not supported for stores using the following domain extensions: `.ru`,`.рф`,`.рус`,`.by`,`.бел`,`.su`.
